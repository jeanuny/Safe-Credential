# Safe-Credential

An App that helps you store all your credentials, logings, and password of any sort locally and in complete security

# How to use

1 - Make sure you have all these dependencies installed:

- cryptography
- base64
- bcrypt
- hashlib
- customtkinter
- tkinter
- pyperclip
- json
- os

2 - Run the app by executing the following command in your terminal:

   ```bash
   python app/app.py
   ```

3 - Follow the on-screen instructions to create an account, log in, and manage your credentials securely.

# Features

- Securely store and manage your credentials locally.
- Encrypt and decrypt passwords using strong encryption methods.
- User authentication with hashed passwords.
- Intuitive GUI built with CustomTkinter for easy navigation.
- View, add, update, and delete credential entries. # Update and Delete features to be implemented in future versions.
- OTP generation. # OTP feature to be implemented in future versions.
